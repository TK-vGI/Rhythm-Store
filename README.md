# Rhythm Store
Learn the basics of SQL. Test yourself in a series of challenges, from simple to complicated. The project uses the SQLite syntax.

Managing an e-commerce site can be complex, involving tracking orders, products, employees, and other vital data.  
While ledger-keeping was an option half a century ago, modern databases have now taken over. Are you interested in  
testing your skills with complex queries using an e-commerce database?

## Stage 1/6: Duplicate accounts
Delete users registered with the same email.  
## Stage 2/6: Inactive customers
Find the users who have never placed an order.  
## Stage 3/6: Best-selling albums
Find the best-selling albums in the database.  
## Stage 4/6: Find playlist duration
Find the playing time of all playlists.  
## Stage 5/6: Best-selling artist
Find the best-selling artists.  
## Stage 6/6: Find fans
Find the fans of the best-selling artist.
